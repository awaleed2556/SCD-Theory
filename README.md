Shit you need:
-Visual Studio any version
-SQL Server Management Studio

How to Run:
-You will need to clone this repo in Visual Studio (Not VS Code), paste repo link in the "Clone Repositry" Section
-Now create a new database in SSMS i will not tell you how to do so.
-After creating the database. Add the database to Server Explorer in Visual Studio and fetch the connection string
-Paste the connection string in appsetting.json
-open Package Manager Console from VS and run the following commands:
  add-migration [give any name]
  update-database
-Assuming you have done all the above operations without any error, you can simply run the code by pressing the play icon which is the run button fyi.
